Level Select 씬 만들기

<오늘의 과제>
1. Level Select 씬  UI 구성
1) Title Text : LEVEL SELECT
2) Level Button : 스크롤 기능 구현
3) 스크롤 바 구현
 
2. 버튼 기능 구현
1) 버튼 애니메이션

2) 버튼 클릭 : - 함수 구현
   Debug.Log ("Level 1 버튼 클릭");
   Debug.Log ("Level 2 버튼 클릭");

2) 씬 이동 구현
- PlayScene -> Level01 씬 이름 변경
- Level01 씬 복사 -> Level02 씬 만들기

3. 레벨 Lock, UnLock
1) 레벨 Lock - 해당 레벨의 선택 버튼이 비활성되도록 만든다
- 모든 버튼 Lock (모든 레벨 Lock)
- 1레벨은 무조건 UnLock

4. Level Lock +
- 모든 레벨 버튼 비활성 
- 1레벨은 무조건 플레이가 가능해야 합니다.
- 유저가 플레이가 가능한 Max 레벨 : nowLevel
- 게임메니저에서 게임 클리어 처리 - nowLevel 저장
//유저 플레이 데이터 저장하기 (Save)
//다음 레벨로 이동하기

5. Level Unlock - 1 Level 클리어시 2 Level Unlock
- 유저가 어느 레벨까지 플레이 했는지 저장하고 있어야 합니다
- 유저가 레벨 클리어 하면 클리어 레벨을 저장해야 합니다
- 게임을 다시 실행하면 플레이 가능한 레벨은 언락되어져 있어야 합니다.
- 1레벨 클리어 하고 2레벨 플레이가 가능하게 합니다
- 다음에 게임을 실행하였을때 레벨 셀럭트에서 2레벨이 선택이 가능해야 합니다.

6. 다음에 게임을 실행했을때 마지막에 언락된 레벨이 맨위에 나오도록 조정
(강제 설정 : 클리어 레벨을 11로 강제 저장한 후 Level 셀렉트에서 12번 레벨 버튼이 상단에 나오도록 한다)
